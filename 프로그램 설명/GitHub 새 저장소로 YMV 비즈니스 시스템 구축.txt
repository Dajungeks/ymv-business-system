새 저장소로 YMV 비즈니스 시스템 구축
Step 1: GitHub에서 새 저장소 생성
GitHub 접속 → "New repository" 클릭
저장소 설정:
Repository name: ymv-business-system
Description: YMV 베트남 한국기업 통합 비즈니스 관리 시스템
Public or Private 선택
✅ Add a README file
✅ Add .gitignore → Python 선택
Create repository 클릭
Step 2: 로컬에 저장소 클론
bash
# 저장소 클론
git clone https://github.com/Dajungeks/ymv-business-system.git
cd ymv-business-system

# 기본 폴더 구조 생성
mkdir -p app/modules/auth app/modules/system app/modules/admin app/shared
mkdir -p database static/css static/js .streamlit

🎯 다음 단계 실행 명령어
위의 파일들을 모두 생성한 후, 다음 명령어들을 순서대로 실행해주세요:

bash
# 1. 저장소 클론 (위에서 생성한 새 저장소)
git clone https://github.com/Dajungeks/ymv-business-system.git
cd ymv-business-system

# 2. 폴더 구조 생성
mkdir -p app/modules/auth app/modules/system app/modules/admin app/shared
mkdir -p database static/css static/js .streamlit

# 3. 파일들 생성 (위의 아티팩트 내용들을 각각 저장)
# requirements.txt
# .env.example  
# streamlit_app.py
# README.md

# 4. Python 환경 설정
python -m venv venv
source venv/bin/activate  # Mac/Linux
# 또는
venv\Scripts\activate     # Windows

# 5. 패키지 설치
pip install -r requirements.txt

# 6. Git 커밋
git add .
git commit -m "Initial project setup with basic structure"
git push origin main
